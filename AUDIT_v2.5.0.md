# Bellwether v2.5.0 audit — Authored Romance & Family

## Evidence before editing
The v2.4.0 tree already contained persistent multidimensional player relationships, explicit social-act consequences, grievances/apologies, shared history, NPC memory/cognition, autonomous NPC life goals, population households, and cottage expansion. It did not contain authoritative player courtship, partnership, cohabitation, or family-intent state.

## Implementation boundary
v2.5.0 adds a bounded authored relationship-life model for Jonah and Mara only. Relationship progression requires existing affinity/trust/familiarity, deliberate shared time, and explicit commitment. Cohabitation requires the existing habitable upper-room cottage expansion. Family life begins as an explicit shared-future decision; this release does not fabricate children, pregnancy, marriage canon, or LLM-authored romance events.

## Preservation
Existing relationship scores, social consequences, NPC autonomy, story canon, cottage/property systems, and dialogue authority remain intact. The new model composes with them rather than replacing them.

## Cleanup
Generated bytecode/cache directories and stale live diagnostic artifacts are removed from the release package. Historical diagnostic source and audit provenance are retained.
