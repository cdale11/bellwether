# Bellwether v0.3.0 Part 2 — Visual Identity and Scene Presentation

Part 2 introduces an asset manifest, state-aware scene resolver, artwork-backed scene stage, semantic warm UI palette, seasonal/daypart/weather atmosphere hooks, responsive scene focal positioning, and preserved side-story presentation in the dedicated Journal.

The approved Bellwether visual direction is soft-painted rural anime with life-sim warmth. Psychological texture is reserved for state-justified later use.

The first approved visual reference ships inside the game package and is resolved for Ashcroft Cottage / Cottage Garden contexts. Locations without approved art use a graceful illustrated-gradient fallback rather than displaying incorrect location artwork.

The resolver never uses artwork to invent world state. Weather and daypart are taken from authoritative state. Character portrait slots remain fallback-ready until canonical standalone character assets are approved.
