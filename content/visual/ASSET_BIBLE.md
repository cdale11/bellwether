# Bellwether Visual Asset Bible — v0.3.0 Part 2

## Locked direction
Soft-painted rural anime is the foundation, with life-sim warmth in ordinary play and textured psychological distortion reserved for justified supernatural or psychological states.

## Canon rules
Recurring buildings must preserve architecture across weather, season, time and horror variants. Core NPCs use consistent canonical appearance and bounded expression families. Background residents use simpler, less-specific representation. Visual state never overrides authoritative simulation state.

## Scene grammar
Base location + seasonal base + daypart treatment + weather treatment + authoritative character presence + event modifier + supernatural modifier.

## Palette grammar
Ordinary daytime uses warm cream, parchment, muted greens, sunlit wood, stone and earth tones. Night darkens naturally. Weather changes atmosphere without implying supernatural horror. Horror corrupts learned normality and may only be driven by authoritative anomaly, psychology or recurrence state.

## Production rule
Approved visual references may ship with the game. Standalone production assets replace references incrementally without changing resolver contracts.
